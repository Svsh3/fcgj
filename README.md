# 🌸 Юки — Telegram AI бот

## Что умеет
- Реагирует на своё имя (Юки, Юку, Юкой, yuki...)
- Помнит контекст разговора (SQLite база данных)
- Модерирует чат по правилам администратора
- Работает 24/7 на Railway.app

---

## 🚀 Деплой на Railway (пошагово)

### Шаг 1 — Подготовка токенов

**Telegram токен:**
1. Напиши @BotFather в Telegram
2. Команда `/newbot`
3. Придумай имя и username боту
4. Скопируй токен вида `123456:ABC-DEF...`

**Gemini API ключ:**
1. Зайди на https://aistudio.google.com
2. Нажми "Get API Key" → "Create API key"
3. Скопируй ключ

---

### Шаг 2 — Загрузка на GitHub

1. Зарегистрируйся на https://github.com
2. Создай новый репозиторий (New repository)
   - Название: `yuki-bot`
   - Публичный или приватный — не важно
3. Загрузи все файлы из этой папки:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.toml`

Можно через сайт GitHub: нажми "Add file" → "Upload files"

---

### Шаг 3 — Деплой на Railway

1. Зайди на https://railway.app
2. Нажми "Start a New Project"
3. Выбери "Deploy from GitHub repo"
4. Подключи GitHub и выбери репозиторий `yuki-bot`
5. Railway сам обнаружит файлы и начнёт сборку

---

### Шаг 4 — Добавить токены (переменные окружения)

В Railway → твой проект → вкладка **Variables** → добавь:

| Название | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от @BotFather |
| `GEMINI_API_KEY` | ключ от Google AI Studio |

После добавления Railway автоматически перезапустит бота.

---

### Шаг 5 — Добавить Volume (для базы данных)

Чтобы память не сбрасывалась при перезапуске:

1. В Railway → твой проект → нажми **"+ Add"** → **"Volume"**
2. Mount Path: `/app`
3. Нажми "Add"

Готово! База данных `yuki_memory.db` будет сохраняться постоянно.

## 🛡️ Команды администратора

| Команда | Что делает |
|---|---|
| `/set_rules текст` | Установить правила чата |
| `/mod_on` | Включить модерацию |
| `/mod_off` | Выключить модерацию |
| `/show_rules` | Показать текущие правила |
| `/clear` | Очистить историю разговора |

### Пример использования:
```
/set_rules Запрещён мат, спам и реклама сторонних ресурсов. Будьте вежливы друг с другом.
/mod_on
```

## ⚠️ Важно для модерации
Чтобы бот мог удалять сообщения, добавь его в чат как **администратора** с правом удалять сообщения.
